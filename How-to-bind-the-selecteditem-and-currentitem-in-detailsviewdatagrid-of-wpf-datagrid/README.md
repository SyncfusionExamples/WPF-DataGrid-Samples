# How to bind the selecteditem and currentitem in DetailsViewDataGrid of WPF DataGrid (SfDataGrid)?

This sample show cases how to bind the selected item and current item in [DetailsVIewDataGrid](https://help.syncfusion.com/cr/wpf/Syncfusion.UI.Xaml.Grid.DetailsViewDataGrid.html) of [WPF DataGrid](https://www.syncfusion.com/wpf-ui-controls/datagrid) (SfDataGrid).

You can bind the `SelectedItem` and `CurrentItem` properties directly in [WPF DataGrid](https://www.syncfusion.com/wpf-ui-controls/datagrid) (SfDataGrid). But in Master DetailsView you canâ€™t be able to bind the SelectedItem or CurrentItem directly in [DetailsViewDataGrid](https://help.syncfusion.com/cr/wpf/Syncfusion.UI.Xaml.Grid.DetailsViewDataGrid.html). You can overcome this by using behavior with dependency properties which is bounded to ViewModel properties. You have to use [SelectionChanged](https://help.syncfusion.com/cr/wpf/Syncfusion.UI.Xaml.Grid.SfDataGrid.html#Syncfusion_UI_Xaml_Grid_SfDataGrid_SelectionChanged) event of corresponding [DetailsViewDataGrid](https://help.syncfusion.com/cr/wpf/Syncfusion.UI.Xaml.Grid.DetailsViewDataGrid.html) to get the `SelectedItem` and `CurrentItem` of [WPF DataGrid](https://www.syncfusion.com/wpf-ui-controls/datagrid) (SfDataGrid).
